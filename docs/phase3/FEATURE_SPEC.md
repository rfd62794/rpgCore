# Phase 3 Feature Specification

## Core Features

### Feature 1: Tower Placement
- Players click grid cells to place towers
- Towers cost gold
- Players select which creature to use as tower
- Creature appearance determined by genetics

### Feature 2: Wave System
- Enemies spawn in waves
- Wave strength scales by game difficulty
- Creatures can survive multiple hits
- Enemies move along path

### Feature 3: Creature Genetics â†’ Tower Appearance
- Tower visual appearance determined by creature genetics
- Traits affect tower stats (range, damage, speed)
- Breeding better creatures = better towers

### Feature 4: Economy
- Players earn gold from wave completion
- Gold buys towers and upgrades
- Genetic traits improve economy (more gold/wave)

### Feature 5: UI/HUD
- Tower info panel (stats, cost, traits)
- Wave counter and status
- Gold counter
- Game over screen (win/lose)

### Feature 6: Creature Feedback Loop
- Send winning creatures back to breeder
- Losing creatures show in graveyard
- Experience/level system optional

## Non-Features (Out of Scope)

- Mobile support (Phase 4)
- Procedural wave generation (Phase 4)
- PvP multiplayer (Phase 4)
- Web deployment (Phase 4)
- Custom creature editor (Phase 5)
